# Moneytree Link iOS SDK - Migration Guide - V5 to V6

The main differences between V5 and V6 are:
  - The format of the frameworks: the frameworks are in `.xcframework` rather than `.framework`.
  - The way how the frameworks are distributed: besides manual integration, the frameworks can now be integrated via Cocoapods and Swift Package Manager.

## Manual integration

- Remove the imported `Moneytree*.framework`s.
- Follow the [manual steps above.](../../README.md#manual)

## Third Party Dependency Management

- Remove the imported `Moneytree*.framework`s.
- You can use either:
  - CocoaPods, follow [these instructions.](../../README.md#cocoapods)
  - Swift Package Manager, follow [these instructions.](../../README.md#swift-package-manager)
